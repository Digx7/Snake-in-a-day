# Snake-in-a-day
Tried making this game in one day, but instead it took three.

You can play the game here: https://digx7.itch.io/snake-in-days 
Play snake alone or with friends on one keyboard!


=========================================

Controls
          Movement
Player 1 - WASD
Player 2 - Arrows
Player 3 - YGHJ
Player 4 - PL;'
